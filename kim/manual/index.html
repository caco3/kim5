Documentation of Kde Image Menu

Kim (Kde Image Menu)

Author: Charles Bouveyron - Webpage: http://bouveyron.free.fr/kim
Kim is a konqueror service menu which allows to resize, convert, rotate, (...) your images without to use a graphical application. This service menu can be considered as a frontend of ImageMagick. Discover now the simplicity of the use of Kim !

Menus:

  • Compress and resize
    This menu allows to compress to different levels the images, resize to different sizes the images.
  • Convert and rotate
    This menu allows to convert the images in JPEG, PNG, GIF, TIFF or many other formats and rotate the images.
  • Treatment and publication
    This menu contains the following actions: rename, send by mail, create a pdf album, convert in grayscale, add an annotation, add a border, create a multiple TIFF image and convert a Multiburst image in a GIF animation.
  • Plugins (optional)
    This optional menu allows to create a web gallery using Kalbum and create a MPEG movie from the given images using MPEGtools.
  • Video Compress
    Does a basic video recompression with ffmpeg to 720p mp4

Installation

  • Unzip the archive,
  • cd kim,
  • run ./install in root.
